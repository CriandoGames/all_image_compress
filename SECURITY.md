# Security Policy

## Versões com Suporte

| Versão  | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reportando uma Vulnerabilidade

Abra uma [Issue](../../issues/new) descrevendo a vulnerabilidade encontrada.

Após o recebimento:
- Você receberá uma resposta em até **7 dias**
- Acompanharemos o progresso e informaremos sobre a resolução
- Vulnerabilidades confirmadas serão corrigidas em um próximo release com crédito ao responsável pelo report (se desejado)
